# SwirlDB Documentation Site

This is the documentation site for SwirlDB, built with [Astro](https://astro.build).

## Development

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:4321`

## Building

```bash
npm run build
```

Output will be in `./dist/`
